Tetris 2012
------------
A Tetris remake.

Controls
---------
F1 - Help.
W / Up - Rotate the figure.
A / Left - Move the figure left.
D / Right - Move the figure right.
S / Down - Move the figure down.
Space - Drop the figure.
T - Turn on/off sound.
H - Turn on/off the figure helper.
F12 - Take a screenshot.

Scoring
--------
Points for 1 line = 10 x level
Points for 2 lines = 25 x level
Points for 3 lines = 50 x level
Points for 4 lines = 100 x level

Notes
------
If there is no sound in the game, install oalinst.exe located in Redist folder.
